# Calculator App

##  Overview

The Calculator App is an Android application developed using Java and XML in Android Studio. It performs basic arithmetic operations while providing a clean and user-friendly interface. The application also includes input validation and calculation history to improve usability.

##  Objective

To develop a calculator application that performs common arithmetic operations with a responsive and intuitive user interface while strengthening Android application development concepts.

##  Features

* Addition, Subtraction, Multiplication, and Division
* Percentage (%) calculations
* Decimal number support
* AC (Clear All) and DEL (Delete) functionality
* Prevents multiple operators in succession
* Prevents multiple decimal points in a single number
* Displays only the final result after evaluation
* Calculation History
* Clean and modern user interface

##  Tools & Technologies

* Android Studio
* Java
* XML
* Android SDK

## 📌 Steps Performed

1. Designed the calculator interface using XML layouts.
2. Implemented button click listeners in Java.
3. Developed arithmetic operation logic.
4. Added input validation to avoid invalid expressions.
5. Implemented calculation history.
6. Tested the application on Android Emulator and physical devices.

##  Outcome

Successfully developed a fully functional calculator application capable of performing arithmetic calculations with enhanced input validation and history support.
